### ✅ 2. GUI按鈕功能修復
**問題**: Spinbox按鈕無法點擊，無反應
**解決**: 從tk.Spinbox改用ttk.Spinbox，添加詳細除錯

**修復措施**:
```python
# 修復前: tk.Spinbox (按鈕無反應)
self.sy = tk.Spinbox(from_=2020, to=2030, value=year)

# 修復後: ttk.Spinbox + 除錯系統
self.sy = ttk.Spinbox(from_=2020, to=2030)
self.sy.set(year)
```

**除錯增強**:
```python
try:
    self.gui.emit("[DEBUG] 🔘 按鈕被點擊")
    self.sy.set(now.year)  # ttk.Spinbox方法

```
