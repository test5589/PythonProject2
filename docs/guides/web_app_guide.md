# 交易機器人 - Web資料視覺化應用

## 🌐 Web應用功能

您的交易機器人現在包含了一個強大的Web資料視覺化界面，用於分析交易資料的完整性和來源分佈。

### 📊 主要功能

#### 1. **資料來源分佈** 📈
- 可視化顯示不同資料來源（real、interpolated、Aggregation等）
- 顏色編碼：綠色=真實資料，橘色=內插資料，黃色=聚合資料
- 支援多個時間框架分析

#### 2. **缺口視覺化** 🕳️
- 直觀顯示資料缺失情況
- 紅色長條表示缺口，灰色表示有資料
- 支援快速時間範圍設定（2、4、6、8、16、24小時）

#### 3. **秒級視覺化** ⏱️
- 分析連續缺失的最大區塊
- 識別最分散破碎的時間段
- 找出最完整的時間段
- 支援1秒、2秒、5秒級別分析

#### 4. **除錯日誌** 🐛
- 即時查看應用運行日誌
- 包含資料庫查詢參數和錯誤信息
- 可清除日誌功能

## 🚀 如何啟動Web應用

### 方法1：使用批次檔案（推薦）
雙擊執行 `start_web_app.bat` 檔案

### 方法2：命令行啟動
```bash
# 在專案根目錄執行
python -m streamlit run web/streamlit_app.py --server.headless true --server.port 8501
```

### 方法3：直接運行Python腳本
```bash
# 在專案根目錄執行
streamlit run web/streamlit_app.py
```

## 🌐 訪問應用

啟動後，在瀏覽器中訪問：
```
http://localhost:8501
```

## 💡 使用提示

1. **側邊欄設定**：
   - 選擇交易對（BTCUSDT、ETHUSDT等）
   - 選擇時間框架（15秒到1天）
   - 設定分析時間範圍

2. **快速時間設定**：
   - 在「缺口視覺化」頁籤中
   - 使用快速按鈕設定2、4、6、8、16、24小時範圍

3. **資料解釋**：
   - **綠色**：即時真實資料（最可靠）
   - **橘色**：內插補齊資料
   - **黃色**：從真實資料聚合而成
   - **黑色**：包含內插資料的聚合

4. **秒級分析**：
   - 僅適用於15秒到1分鐘的時間框架
   - 幫助識別資料收集的穩定性

## 🔧 技術依賴

- **Streamlit**：Web界面框架
- **Plotly**：互動式圖表
- **Pandas**：資料處理
- **SQLite**：資料存儲

## 📝 注意事項

- 確保資料庫文件存在：`data/system_data.db`
- 應用會在後台運行，不會自動開啟瀏覽器
- 如需停止應用，按 `Ctrl+C` 或關閉命令行窗口
